This course teaches you to make informed decisions by dividing deep into real concepts, architecture, and unidirectional dataflow. You will learn how to bootstrap an application and how to build pages and reusable elements using Angular components and the new Angular syntax. You will also learn about the basics of how Angular works, and why Angular has emerged as a popular framework for JavaScript/Typescript Application Development. You will also learn how to properly setup your development environment for creating an Angular app, including installing VS Code, Node.JS, Typescript, and the Angular CLI.

You will be able to understand the programming model provided by the AngularJS framework. In addition, you will learn the necessary skills needed to create single page web applications using the MVC pattern of AngularJS. Lastly, you will be knowledgeable enough to understand Angular controllers and directives, control Angular data bindings, and implement Responsive Web Applications with AngularJS;

Our professor for the Professional Domain Course 20 (PDC20) will be Mr. Fernand T. Layug, MIT. He is a graduate of Bachelor of Science in Information Technology and Master of Science in Information Technology. Given his compelling credentials, you will be assured of quality education.
